# 📋 מדריך סנכרון תוכניות אימון — מערכת מושלמת

## 📌 מה שנשתנה

### 1. **Validation יום-קבוצה** ✅
כשהבוט מציע לשמור תוכנית, הוא **בודק**:
- האם הקבוצה מתאמנת באותו יום בסניף זה?
- אם לא → תן **אזהרה ברורה** לפני השמירה

**דוגמה:**
```
⚠️ גן חובה לא מתאמנת בסירקין ביום שני!
קבוצות שמתאמנות: ד-ו, ג, א-ב, ז-בוגרים
```

### 2. **Logging מלא** 📝
כל שמירה תוכנית לוגגה ל-`~/logs/training_plans_sync.log`:

```
[2026-07-20 21:15:30] 🟢 START: save_full_day(סירקין, 2026-07-20)
[2026-07-20 21:15:30]   קבוצות בתוכנית: ד-ו, ג, א-ב, ז-בוגרים
[2026-07-20 21:15:30]   טאב בגיליון: 'סירקין' (ID: 123456)
[2026-07-20 21:15:30]     > קבוצה: ד-ו | טאב: סירקין | תא: E
[2026-07-20 21:15:30]       שורה 15: חימום = ריצה 3:00 + גלגולים
[2026-07-20 21:15:30]       שורה 16: תרגול = יוקו שיו גטמה 3×3:00
[2026-07-20 21:15:30]     ✓ 2 תאים עודכנו בגיליון
[2026-07-20 21:15:30]       ✓ עיצוב רענן
[2026-07-20 21:15:30]       ✓ ארכיון שמור

[2026-07-20 21:15:30] ============================================================
```

**מקום הלוג:**
```bash
~/logs/training_plans_sync.log
```

**איך לקרוא את הלוג:**
```bash
# ראה את 50 השורות האחרונות:
tail -50 ~/logs/training_plans_sync.log

# ראה רק שגיאות:
grep "❌\|⚠️" ~/logs/training_plans_sync.log

# ראה סיכום (START/SUCCESS/PARTIAL):
grep "START\|SUCCESS\|PARTIAL\|ERROR" ~/logs/training_plans_sync.log
```

### 3. **Error Handling טוב** 🛟
בכל שלב של הסנכרון:
- ✅ קבוצה נשמרה בהצלחה
- ⚠️ אזהרה (אין תוכן לקבוצה)
- ❌ שגיאה ברורה (קבוצה לא קיימת בטאב)

**בטלגרם:**
```
✅ ד-ו
✅ ג
⚠️ א-ב: אין תוכן
❌ ז-בוגרים: שגיאה בשמירה — תא לא נמצא
```


## 🎯 מה לבדוק אם משהו לא עובד

### 1. **הבוט לא מוכן לשמור**
```
❌ "לא נמצאת תוכנית לשמירה"
```
→ שלח את התוכנית מחדש

### 2. **אזהרה על קבוצה**
```
⚠️ ד-ו לא מתאמנת בסירקין ביום שלישי!
```
→ זה תקין! בודקים שהקבוצה בפועל מתאמנת ביום הזה

### 3. **שגיאה בשמירה**
```
❌ שגיאה בשמירה: קבוצה 'א-ב' לא נמצאה בלשונית סירקין
```
→ בדוק את הלוג:
```bash
tail -20 ~/logs/training_plans_sync.log
```

### 4. **חלק מהקבוצות לא נשמרו**
```
⚠️ א-ב: אין תוכן
```
→ Claude לא כתב תוכן לקבוצה הזו. נסה לערוך או לשלוח מחדש

## 📊 GROUPS_BY_DAY — מה בדיוק נבדק

הבוט בודק על פי הטבלה הזו (בbot.py):

```python
GROUPS_BY_DAY = {
    "סירקין": {
        0: ["ד-ו", "ג", "א-ב", "ז-בוגרים"],  # שני — ללא גנים!
        3: ["ד-ו", "ג", "א-ב", "גן חובה", "ז-בוגרים"],  # חמישי — עם גנים
    },
    "חגור": {
        6: ["ד-ח", "א-ג", "גנים"],  # ראשון
    },
    # וכו'...
}
```

**ערכים:**
- `0` = שני
- `1` = שלישי
- `2` = רביעי
- `3` = חמישי
- `4` = שישי
- `6` = ראשון

```python
GROUPS_BY_DAY = {
```
